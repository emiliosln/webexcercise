# Webexcercise

## Prerequisites

Python or Docker

## Install

From Docker:
```
docker run -p 5000:5000 emiliosln/webexcercise:johnnyrotten
```

Local install:
```
git clone https://github.com/emiliosln/webexcercise.git
cd webexcercise
pip install -r requirements.txt
python web.py
```

## RUN

Open http://localhost:5000 in a browser
